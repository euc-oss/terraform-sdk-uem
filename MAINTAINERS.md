# Maintainers

This Go SDK is maintained by the Omnissa Workspace ONE UEM engineering team.

## Current maintainers

## Response expectations

GitHub issues filed against this repository are reviewed on a best-effort
basis by the maintainer team. There is no formal SLA for issue response or
fix turnaround at this time. Issues that include clear reproduction steps,
SDK and Go versions, and deployment context (cloud vs. on-prem) are easier
to triage and tend to move faster.

## Production support

If you need production support for the SDK or for Workspace ONE UEM itself,
please open a Support Request via [Omnissa Customer Connect](https://customerconnect.omnissa.com/home).
GitHub issues are not a substitute for paid support.

## Becoming a maintainer

This SDK is generated and released from an internal source pipeline.
Maintainership is currently restricted to Omnissa employees who own that
pipeline. The maintainer roster is reviewed periodically.
